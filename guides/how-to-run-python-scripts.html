How to Run Python Scripts

A beginner guide for Windows, Mac, and Linux. Install Python, open a terminal, install dependencies, and run your first script in under 10 minutes.

All tools on this site are Python scripts. This guide shows you exactly how to get them running, even if you've never used Python before.

Step 1 - Install Python

First check if Python is already installed. Open a terminal (see Step 2) and run:

python --version

If you see Python 3.x.x you're good. If not, install it:

  • Windows - Download from python.org/downloads. During install, check "Add Python to PATH".
  • Mac - Install via Homebrew: brew install python3. Or download from python.org.
  • Linux - Usually pre-installed. If not: sudo apt install python3 python3-pip

Step 2 - Open a terminal

  • Windows - Press Win + R, type cmd, press Enter. Or search "Command Prompt" in the Start menu.
  • Mac - Press Cmd + Space, type "Terminal", press Enter.
  • Linux - Press Ctrl + Alt + T.

Step 3 - Clone or download the script

Every tool on this site is on GitHub. You have two options:

Option A - Clone with git (recommended):

git clone https://github.com/Get-Ai-Tools/reddit-scraper
cd reddit-scraper

Option B - Download as ZIP: Go to the GitHub page, click the green "Code" button, then "Download ZIP". Unzip it, then navigate to the folder in your terminal:

# Windows
cd C:\Users\YourName\Downloads\reddit-scraper

# Mac / Linux
cd ~/Downloads/reddit-scraper

Step 4 - Install dependencies

Every tool comes with a requirements.txt file listing the libraries it needs. Install them all with one command:

pip install -r requirements.txt

This may take a minute. You'll see packages being downloaded and installed.

Step 5 - Set your API key

Most tools need a Claude API key for the AI features. The safest way is an environment variable so the key never ends up in your code.

Windows (Command Prompt):

set ANTHROPIC_API_KEY=sk-ant-your-key-here

Mac / Linux:

export ANTHROPIC_API_KEY=sk-ant-your-key-here

Or create a .env file in the project folder:

ANTHROPIC_API_KEY=sk-ant-your-key-here

The scripts on this site will automatically load .env files via python-dotenv.

Don't have a Claude API key? Get one free at console.anthropic.com. Many tools work without any API key at all.

Step 6 - Run the script

Each tool's page shows the exact command to run. Generally it looks like:

python script.py --option value

For example, the Reddit scraper:

python scraper.py --subreddit python --sort top --limit 50

Common errors and fixes

"python is not recognized" (Windows)

Python wasn't added to PATH during install. Either reinstall Python and check "Add Python to PATH", or try python3 instead of python.

"ModuleNotFoundError: No module named X"

A dependency isn't installed. Run pip install -r requirements.txt again. If a specific module is missing, run pip install module-name.

"pip is not recognized"

Try pip3 instead of pip. On Windows, try python -m pip install -r requirements.txt.

"Permission denied" (Mac/Linux)

Prefix with sudo: sudo pip install -r requirements.txt. Or better, use a virtual environment (see below).

Optional: Use a virtual environment

Virtual environments keep each project's dependencies separate. Good practice for serious use:

# Create a virtual environment
python -m venv venv

# Activate it (Mac/Linux)
source venv/bin/activate

# Activate it (Windows)
venv\Scripts\activate

# Now install dependencies
pip install -r requirements.txt

# When done, deactivate
deactivate
