How to Use the Claude API with Python

Get your free Claude API key, install the Python SDK, and run your first AI-powered script in under 10 minutes. This is everything you need to run any tool on getaitools.dev.

Step 1 — Get a free API key

  1. Go to console.anthropic.com
  2. Sign up for a free account
  3. Go to API Keys → Create Key
  4. Copy the key — it starts with sk-ant-

New accounts get free credits to start. After that, usage is pay-as-you-go. The cheapest model (claude-haiku) costs about $0.001 per 1,000 words — most scripts cost fractions of a cent to run.

Step 2 — Install the SDK

pip install anthropic

Step 3 — Run your first script

import anthropic

client = anthropic.Anthropic(api_key="sk-ant-YOUR_KEY_HERE")

message = client.messages.create(
    model="claude-haiku-4-5-20251001",
    max_tokens=200,
    messages=[
        {"role": "user", "content": "Summarize what the Claude API is in 2 sentences."}
    ]
)

print(message.content[0].text)

Run it with python script.py and you'll get a response in seconds.

Step 4 — Use it with any tool on this site

Every tool on getaitools.dev uses the same pattern. You pass your API key with --api-key sk-ant-...:

# Example with the Reddit Scraper
python scraper.py --api-key sk-ant-... --subreddit python --summarize

# Example with the PDF Summarizer
python summarizer.py --api-key sk-ant-... --pdf report.pdf

# Example with the Meeting Summarizer
python summarizer.py --api-key sk-ant-... --transcript meeting.vtt

Which model should I use?

All tools on this site default to claude-haiku-4-5-20251001 — it's the fastest and cheapest model, and it's more than capable for summarization, classification, extraction, and generation tasks.

Model Speed Cost Best for
claude-haiku-4-5-20251001 Fastest Cheapest Bulk tasks, classification, extraction
claude-sonnet-4-6 Fast Mid Complex writing, analysis
claude-opus-4-6 Slower Higher Most complex tasks

Keep your API key safe

Never hardcode your key in a script you'll share. Instead, use an environment variable:

# Set it once in your terminal
export ANTHROPIC_API_KEY="sk-ant-..."

# Then in Python
import os, anthropic
client = anthropic.Anthropic(api_key=os.environ["ANTHROPIC_API_KEY"])
