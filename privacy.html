Privacy Policy

Last updated: April 14, 2026

The short version

GetAITools.dev does not collect, store, or sell your personal data. The tools on this site are open source scripts you download and run on your own machine. We never see your API keys, your prompts, or your outputs.

What we collect

This website collects minimal data through standard web infrastructure:

  • Cloudflare analytics - We use Cloudflare Pages to host this site. Cloudflare may collect anonymized traffic data (page views, country, browser type) without storing personally identifiable information. See Cloudflare's privacy policy.
  • No cookies - We do not set any tracking cookies.
  • No accounts - We do not offer user accounts or collect email addresses.

The tools themselves

Tools are available in two forms:

  • Python scripts (GitHub) - Run entirely on your local machine. Your API keys and data never pass through our servers. API calls go directly from your machine to Anthropic.
  • Browser tools - Your API key is stored in your browser's localStorage only. When you run a tool, your browser sends your key and input to a thin Cloudflare proxy function that immediately forwards the request to Anthropic and returns the result. Nothing is logged or stored on our end. The proxy source code is publicly available on GitHub.

Third-party services

This website uses Google Fonts for typography, which may log your IP address when loading font files. We may also display advertisements through third-party ad networks in the future; any such ads will be disclosed clearly.

Contact

Questions about this policy? Open an issue on our GitHub.
